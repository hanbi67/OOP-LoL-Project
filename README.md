# OOP-LoL-Project
### 1️⃣ 1회차 과제 - 수행한 미션

#### [필수 미션]

- 챔피언을 기반으로한 클래스 2개 생성 후 상태(필드변수) / 행위(메서드) 지정하기 (캡슐화)
- 유사한 성격을 가진 메서드를 모아 추상 클래스로 묶어보기(추상화)
- 기존 클래스에 상속 후 코드 수정하기(상속)
- 다형성을 활용하여 메서드 설정 및 main함수에서 모의 전투 롤플레잉 (다형성)

#### [추가 미션]
- 가렌에게 “레벨업(levelUp)” 메서드를 만들어 체력·공격력 증가하게 하기
- takeDamage 후 체력이 0 이하가 되면 “(챌린저) 사망!” 메시지 출력
- attackDamage 값을 난수(랜덤)로 만들어 “치명타 개념” 추가
- Lux, Teemo 등 챔피언 2명 더 만들기
- 인터페이스를 2개 이상 조합한 챔피언 만들기(Lux)
- Q/W/E/R 스킬을 모두 추상 메서드로 확장해보기

### 2️⃣ 2회차 과제 - 수행한 미션

#### [필수 미션]
- 챔피언 기본 스탯을 GameConstants의 static final 상수로 분리하여 관리
- 전투 발생 시마다 증가하는 static 전투 횟수 카운트 기능 구현
- 모든 챔피언이 동일한 규칙으로 부활하도록 final 메서드로 고정

#### [추가 미션]
- 챔피언별 부활 조건을 추상 메서드로 분리하여 확장 가능하게 설계
- 전투 로그 출력을 static 중첩 클래스로 분리하여 책임 명확화
- static 변수 남용 시 발생하는 문제를 직접 실험하여 확인 (TestStatic class)
- 체력·공격력·방어력 상수를 enum + 상수 조합으로 구조화
- 부활 로직을 템플릿 메서드 패턴으로 확장하여 부가 효과를 다형적으로 구현
